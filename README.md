# Horse Race Game
This porject uses Vue 3 as it's the modern version with better support with everything such as Pinia

## Horse Generation
While each horse gets a color the color word is aslo given for color blind support

## Race Logic
Condition overall gives a boost to the winrate but it's effect on the speed of the horses is lowered at the first and last 250 m of the race making the start and the end of the races and the shorter races overall more unpredictable
